Stock Market predictors

This web app currently contains two models for predicting future stock prices, you can read about the models below before selecting one for use. These models' results should not be used for financial decisions as they are only predictions and are not always accurate.

Prophet Model

The prophet model is mode from the Facebook produced FBProphet python package.This is the description of the model from their website; "Prophet is a procedure for forecasting time series data based on an additive model where non-linear trends are fit with yearly, weekly, and daily seasonality, plus holiday effects. It works best with time series that have strong seasonal effects and several seasons of historical data. Prophet is robust to missing data and shifts in the trend, and typically handles outliers well." I am using this model to create a benchmark to compare a specialized LSTM model (read below) to determine if a neural net model can outperform seasonal trend predictions.

LSTM Model

This model is a neural net powered by the python Keras package that I have configured to predict future stock prices. If you are curious about how the LSTM works you can read more about it here. The model consists of 3 layers, 2 LSTM layers with TAHN activation followed by a dense layer with linear activation.
